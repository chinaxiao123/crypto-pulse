# 💓 CryptoPulse

> **加密市场行情分析CLI工具** — 实时行情、数据分析、一键报告

[![Python](https://img.shields.io/badge/Python-3.10+-blue)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green)](LICENSE)
[![CoinGecko](https://img.shields.io/badge/API-CoinGecko-orange)](https://www.coingecko.com)

---

## ✨ 功能

```
🚀 crypto pulse BTC        → 比特币实时行情
📊 crypto analyze ETH      → 以太坊深度分析
📝 crypto report           → 生成Markdown日报
🎭 crypto post             → 自动发帖到币安广场（配合八手玫瑰🌹）
```

## 🛠️ 安装

```bash
pip install crypto-pulse
# 或者
git clone https://github.com/chinaxiao123/crypto-pulse.git
cd crypto-pulse
pip install -r requirements.txt
```

## 🚀 使用

```bash
# 查 BTC 实时价格
crypto pulse BTC

# 分析 ETH 市场
crypto analyze ETH

# 生成日报
crypto report

# 币安广场自动发帖
crypto post --persona 八手玫瑰
```

## 📦 技术栈

- **Python** + **Typer** — CLI框架
- **CoinGecko API** — 免费行情数据
- **Rich** — 终端美化输出
- **Markdown** — 报告生成
